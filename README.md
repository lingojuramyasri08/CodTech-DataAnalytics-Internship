# Task 4 - Sentiment Analysis using NLP

## Objective
Perform sentiment analysis on text data to classify sentiments as Positive, Negative, or Neutral.

## Technologies & Libraries Used
- Python
- TextBlob
- VADER Sentiment Analyzer

## Work Performed
- Took user text input
- Applied text preprocessing
- Performed sentiment analysis
- Generated polarity and sentiment scores
- Compared TextBlob and VADER outputs

## Preprocessing Techniques Used
- Lowercase Conversion
- Whitespace Removal using `strip()`

## Sentiment Analysis Models
### 1. TextBlob
Used for basic polarity-based sentiment analysis.

### 2. VADER Sentiment Analyzer
Used for improved sentiment analysis on emotional and social-media-like text.

## Key Insights
- TextBlob works well for simple sentences
- VADER handles emotional context more effectively
- Basic sentiment models may fail for sarcasm and complex expressions
- NLP models differ in performance and accuracy

## Learning Outcome
This task helped in understanding:
- Natural Language Processing (NLP)
- Text Preprocessing
- Sentiment Classification
- Model Comparison
- Real-world AI limitations
